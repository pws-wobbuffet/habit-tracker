**Habitus** is an open-source, mobile-first progressive web app for tracking daily and weekly habits. It runs entirely in the browser, installs to your home screen, and is designed to feel like a native iOS app.

<CardGrid>
  <Card title="Offline-first" icon="open-book">
    All data lives in IndexedDB in your browser. No account, no server, no data leaves your device
    by default.
  </Card>
  <Card title="Installable" icon="star">
    Add to your iPhone or Android home screen via Safari or Chrome. Opens in standalone mode with no
    browser chrome.
  </Card>
  <Card title="Pluggable storage" icon="setting">
    Start with IndexedDB. Switch to SQLite WASM or a remote REST server via a single environment
    variable.
  </Card>
  <Card title="MIT licensed" icon="github">
    Clone, fork, and self-host freely. No telemetry, no subscriptions.
  </Card>
</CardGrid>

## Features

- **Today view** -- see habits due today, mark them done, track your week at a glance
- **Habit creator** -- pick a name, emoji icon, and a daily or day-of-week schedule
- **Streaks** -- per-habit and overall streaks based on consecutive-day completion logic
- **Completion notes** -- attach a text note to any completion via the bottom sheet
- **Voice memos** -- record audio directly on any completion entry
- **Calendar heatmap** -- year view showing completion intensity by date
- **Analytics** -- lifetime totals, streak summary, and active day count
- **Achievements** -- milestone badges for streaks, totals, and consistency
- **Settings** -- display name and theme preferences
